Muse IO
========================
A Muse driver for Mac and Windows desktop environments that sends out Muse data as OSC or LSL messages which other programs can interface with.

You can access tutorials and on-line documentation for MuseIO here:
https://sites.google.com/a/interaxon.ca/muse-developer-site/museio


Muse Lab
========================
A visualizer tool for brainwave and other types of data. This receives data from MuseIO and allows you to record and annotate data, filter it through various algorithms.

You can read Muse Lab tutorial here:
https://sites.google.com/a/interaxon.ca/muse-developer-site/muselab/tutorial


Muse Player
========================
A command-line desktop program which allows you to receive Muse data in one format and output it in one or more formats.

Read about Muse Player here:
https://sites.google.com/a/interaxon.ca/muse-developer-site/muse-player


libmuse android
========================
A library for interfacing with Muse headbands, including finding
paired Muses, connecting to them, reading their state, and receiving packets
for raw EEG data and all other values. You can use it in your own applications,
subject to the terms of our license.
